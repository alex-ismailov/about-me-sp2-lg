About me

I got carried away with programming in 201 ... year, for the sake of interest I enrolled in courses in the C language as a specialist, took two courses - introduction to programming and the basics of the C language. Then I studied Java-rush as a Java developer, and at the same time studied the books Javanjnjn Orelly and the philosophy of Java

Then the mining boom began, I got a rig, so I first got acquainted with the Linux Mint OS. I wrote a console utility (bash script) that monitors my maps in real time, informs me about telegram problems and reboots the rig.

Then I met the person I was running the farm. I set up Linux OS on the rigs on which my script bot was running, and I also monitored the technical state of the farm. Those. in fact, thanks to my script, I have a stable income in addition to the income from my rig.

By the time the Maninig bubble burst, I already began to understand the purpose of the various languages ​​and the environments for which they are intended. In the end, I chose Web development. I clearly understood that I had to go to study at an educational institution.

The first place I got to was GeekBrains, a fullstack JS developer profession. In parallel, I learned from ninja books, yy6, rhino patterns Then for about six months I studied independently PHP on Koterov and WordPress. Unfortunately, I didn't immediately realize that I was only learning the syntax of languages. I started digging deeper, so the SICP book fell into my hands.

While looking for solutions to exercises from sicp, I accidentally found the Hexlet training platform, on which I study to this day.
